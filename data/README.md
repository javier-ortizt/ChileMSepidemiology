This repository does not distribute raw or restricted input data.

Expected local-only inputs used by the analysis scripts include:

- `data/raw/BBDD EM.xlsx`
- `data/raw/fonasa2010-2022.xlsx`
- `data/raw/BeneficiariosFONASA_2023.csv`
- `data/raw/BeneficiariosISAPRE_2023.xlsx`
- `data/population/estimaciones-y-proyecciones-2002-2035-comunas.xlsx`
- `data/population/beneficiariosTramoEtario.xlsx`
- `data/population/Previsionporregion.xlsx`
- `data/geographic/Regional.shp` and associated shapefile sidecars

The public GitHub version keeps only code and paper-facing aggregate outputs in `results/`.

If you have lawful access to the underlying datasets, place them under the same local paths above to rerun the full pipeline.
